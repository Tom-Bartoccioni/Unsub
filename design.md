1. The Main Dashboard (The "Control Center")
   This is the first screen the user sees. It balances the visual data of the graph with the utility of the list.

The Donut Chart: A thick, minimalist circular ring sits at the top. The center of the ring displays the total monthly cost in a large, bold font (e.g., $134.50). The segments of the ring are color-coded by category (Entertainment, Productivity, Wellness).

The Sorted List: Below the graph, subscriptions are listed in cards. They are automatically sorted by renewal date (the one renewing soonest is at the top).

Currency Unity: Every price in the list is shown in the same currency to ensure the total in the graph makes sense.

The Floating "+" Button: A prominent, colorful circular button is positioned in the bottom right corner, ready for new entries.

2. The Detailed Sub-View (The "Deep Dive")
   When a user taps on a subscription (e.g., Netflix), a sleek modal slides up from the bottom.

Historical Data: It shows exactly how much the user has paid in total since they started using the service (e.g., "Total spent since 2022: $416.00").

Timeline: It displays the "Start Date" and the exact "Next Billing Date".

Ghost Toggle: A specific button allows the user to mark this subscription as "to be cancelled," moving it to a separate "Ghost" list for tracking savings.

3. The "Add" Flow (The "Onboarding")
   Clicking the + button triggers a clean, two-step transition.

Step 1: The Library: A grid of recognizable minimalist icons for popular services (Netflix, Spotify, ChatGPT, Gym, etc.). Each icon has a "Standard Price" hint below it to speed up entry.

Step 2: The Custom Creator: If the service isn't in the library, a simple form appears. The user enters the name, selects an icon from a library, sets the price, and chooses the frequency (Monthly/Yearly).
